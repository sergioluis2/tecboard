# TecBoard — Landing Page Responsiva

Landing page desenvolvida como projeto prático do curso de HTML e CSS da
formação Desenvolvedor Front-End da Alura.

## 🔗 Demo

[Acessar a landing page ao vivo](https://sergioluis2.github.io/tecboard/)

## 🎯 Objetivo do projeto

Aplicar fundamentos de HTML semântico e CSS responsivo na construção
de uma landing page funcional, com foco em layout adaptável a diferentes
tamanhos de tela (desktop, tablet e mobile).

## 🛠 Tecnologias

- HTML5
- CSS3
- Media Queries para responsividade
- Git e GitHub para versionamento e deploy

## 📚 O que aprendi

- Estruturação semântica de páginas com HTML5
- Uso de Flexbox para layouts flexíveis
- Aplicação de Media Queries para adaptação responsiva
- Importação e uso de fontes externas via @font-face
- Publicação de site estático via GitHub Pages
